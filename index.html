Corporate Network Tips

If all proxies show "blocked" after a connectivity test:
① Ask IT to whitelist one of the proxy domains above, or
② Deploy a CORS proxy on an internal host (even http://localhost:9999 works) and add it as Custom.
The browser already uses your system/corporate proxy for all outbound requests — Pulse can't bypass that, but a whitelisted CORS relay solves it.
